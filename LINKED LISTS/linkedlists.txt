What is a Linked List?
- A linked list is a form of sequential collection and it does not have to in order.
- A linked list is made up of independent nodes that may contain any data and each node has a reference to the next node in the link.

Types of Linked Lists

1. Singly Linked List
2. Doubly Linked List
3. Circular Linked List
4. Circular Doubly Linked List


Insertion to Linked List in memory

1. At the beginning of the linked list
2. After a node in the middle of the linked list
3. At the end of the linked list
